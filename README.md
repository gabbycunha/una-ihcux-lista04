## Try-Catch
Try-catch é uma estrutura usada para lidar com erros sem fazer o programa parar de funcionar.
O try-catch impede que o programa quebre, permite tratar o erro e melhorar a experiência do usuário
## Pergunta de Reflexão (IHC): 
> "Quando o programa 'crasha' e fecha sozinho por um erro de digitação, qual é o sentimento do usuário? Como o tratamento de erros (mensagens amigáveis) ajuda na confiança que o usuário tem no software?"

O sentimento do usuário é de frustração e com o tratamento com mensagens amigáveis ajuda na experiência do usuário a fazer com que ele não desista do programa facilmente.
